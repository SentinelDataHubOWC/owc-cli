OWCLI
========
Command line interface for OWC component developers

# Introduction
OWCLI is a command line tool to help the OWC component developers.

# How to install
```
cd owc-cli
pip3 install .
```

# How to launch
```
owcli [OPTIONS] COMMAND [ARGS]...
```
e.g.
```
owcli bower --dev
```

# How to show help
```
owcli --help
```
